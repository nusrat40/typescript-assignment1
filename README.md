# TypeScript Assignment 1

This repository contains solutions for a TypeScript assignment focusing on:

- Core TypeScript concepts: types, interfaces, generics, union types, and utility types.
- Object-oriented programming: classes, inheritance, and methods.
- Problem-solving exercises: functions for arrays, strings, and objects.
- Advanced TypeScript features: conditional types, mapped types, and type constraints.

## File Structure

- `solutions.ts` – All coding problems are implemented in a single TypeScript file.
- `blog-1.md` – First blog post (concept explanation with examples).
- `blog-2.md` – Second blog post (concept explanation with examples).
- `README.md` – Project overview and instructions.
